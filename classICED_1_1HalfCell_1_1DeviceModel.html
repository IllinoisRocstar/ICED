Detailed Description

Class to hold the application interface for the electro-chemical device solver.

The device model is the base interface for evaluating the "stateless model" This class provdes the interface between our problem to be solved (in this case an electrochemical device) and the Trilinos solvers.

The model has an interface to a transient solver, which in turn has an an interface to linear (steady-state) solver.
