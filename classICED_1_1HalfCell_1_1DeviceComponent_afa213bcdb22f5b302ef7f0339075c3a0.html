int ApplyInternalBC ( )

Apply internal boundary conditions.

Loop over all the solution variables and apply the internal boundary condition. This averages the solution and function evaluation by the number of overlapping grid points (in this case 2 for 1-D). This is applied to every solution variable that has an internal boundary set. This should be applied only once during a call to the component solve.

Returns
an error code
