Compute the device level evaluation function.

This function is our interface to the Trilinos solvers and essentially drives the solution process

  1. copies the Trilinos single component global device solution to the Trilinos multivector storage at the device level
  2. copies the data from the device to the components
  3. initiates the component solve
  4. copies the data from the components back to the device
  5. copies the function evaluation from the Trilinos global device multivector to the Trilinos global device vector and facilitates any solver specific computations (i.e. implicit transient solve for backward Euler)
Parameters
xInput solution vector
fOutput evaluation function
Returns
An error code
Test:
none
