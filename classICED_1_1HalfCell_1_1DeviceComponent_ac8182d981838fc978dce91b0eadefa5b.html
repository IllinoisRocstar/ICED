int ApplyComponentBC ( )

Apply the appropriate component boundary conditions.

Use the boundary conditions implemented by each physics module to set the component boundary conditions for the component level solution matricies.

Only call boundary condition routines for physics modules that have a valid variableIndex. This indicates that the equations solved by the physics module are included in the global solution matrix and excludes modules that calculate spatially varying material properties

Returns
an error code
