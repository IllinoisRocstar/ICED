Detailed Description

Implementation of the batterySim program.

This object encapsulates the serial program. It inherits from the IRAD::Global::Program type.

The program simulates a half-cell battery. The command line usage goes: 

       batterySim Usage:

       batterySim [-h] [-v [arg] -o <filename> ] <input> 

       -h,--help
          Prints this long version of help.

              -v,--verbosity [arg]

       -o,--output <filename>
          Specifies the name of the output file.

              <input>
          Specifies the input <filename> (required)
