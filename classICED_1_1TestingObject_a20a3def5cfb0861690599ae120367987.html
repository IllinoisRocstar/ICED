Test for ICED::DeviceData.

Parameters
resultICED::TestResults object to store test results.

This function implements builds a DeviceData object and tests a couple of I/O features, specifically that input is passed from the device to the component level and can be overwritten at the component level

There is a test script to copy the input files to IO_Unit_Test. The test script returns the results of the IOCopy test. This test will not pass if IOCopy does not pass.
