Detailed Description

Class that provides the component level interface between the electro-chemical device solver and the physics modules.

This class provides the interface between the global solution data and the physics modules. It contains all the data specific to a electrochemical device component, including a vector of pointers to enabled physics modules and the component boundary conditions. Solution and parameter data storage is provided by the SolutionData object.

A component is a specific piece of the electrochemical device, i.e. a current collector, electrode, separator, etc. The actual make-up of the components is defined by the physics modules enabled for each component.
