void Test__SolverBENL ( ResultsType &  result)
inline

System level transient solver test.

Parameters
resultICED::TestResults object to store test results.

This is a system level test of a transient solver using a Backward Euler method This function builds a DeviceData object, initializes the data, and call the solution routines.

This test uses a single physics module (Diffusion) to solve the heat equation. The device is made up of 3 components in the standard anode-separator-cathode configuration. This utilizes insulated external boundaries with internal free boundaryies at the anode/cathode-separator interface.

Currently compare the solution to the analytic form for a nonlinear heat transfer coefficient.
