Test for ICED::ComponentBoundaryConditions.

Parameters
resultICED::TestResults object to store test results.

This function builds a DeviceData object and tests the application of boundary conditions at the component level.

This test loads the testModule PhysicsModule and does not exhaustively test every boudary condition for every implemented module. It is left to the unit tests for each module to check it's BC implementation.

Input is specified via Parameter objects. A test module is loaded that sets solution data to be a simple list of numbers based on the temperature parameter. Boundary conditions are applied for current collectors and separators. The expected results are verified.
