Detailed Description

Class to hold the solution data for the electro-chemical device solver.

This class contains all the global solution data used to produce a solution for an electrochemical device. It creates an arbitrary number of components that model pieces of the device, i.e. a current collector, electrode, separator, etc. The actual make-up of the components is defined by the physics modules enabled for each component.
