System level transient solver test.

Parameters
resultICED::TestResults object to store test results.

This is a system level test of the kinetics using a Backward Euler method This function builds a DeviceData object, initializes the data, and call the solution routines.

This test uses a single physics module (ButlerVolmer) to solve the kinetics rate equation. The device is made up of 1 component with current collectors on both sides. We set a constant voltage on both ends.
