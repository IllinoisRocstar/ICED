inline

Test for ICED::DeviceDataMapping.

Parameters
resultICED::TestResults object to store test results.

This function builds a DeviceData object and tests the mapping data generated by physics module at the component level to the device level. We scale that data and propagte back to the component level.

Input is specified via Parameter objects. A test module is loaded that sets solution data to be a simple list of numbers based on the temperature parameter. The results are compiled across all components.
