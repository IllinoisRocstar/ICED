System level transient solver test.

Parameters
resultICED::TestResults object to store test results.

This is a system level test of a transient solver using a Backward Euler method This function builds a DeviceData object, initializes the data, and call the solution routines.

This test uses a single physics module (OhmsLawElectrolyte) to solve the heat equation. The device is made up of 3 components in the standard anode-separator-cathode configuration. This utilizes an insulated external boundaries at the anode and cathode with internal free boundaryies at the anode/cathode-separator interface.

We set the initial potential in the electrolyte to 10 in the cathode/anode and 100 in the separator
