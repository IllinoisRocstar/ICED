inline

System level transient solver test.

Parameters
resultICED::TestResults object to store test results.

This is a system level test of a transient solver using a Backward Euler method This function builds a DeviceData object, initializes the data, and call the solution routines.

This test uses a single physics module (OhmsLawSolid) to solve the heat equation. The device is made up of 1 component with current collectors on both sides. We set a constant voltage at one end and a current at the other.

Currently compare the solution to the analytic form for a constant heat transfer coefficient.
