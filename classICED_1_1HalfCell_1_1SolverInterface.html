Detailed Description

Class to hold the interface to the Trilinos NOX Solver.

SolverInterface defines the interface between NOX and the nonlinear problem to solve. Its constructor accepts a PDEProblem object, which it uses to update the Jacobian and compute F(x). This interface is a bit crude; for example, it does not provide a PrecMatrix or Preconditioner.

Todo:
This class currently implements a Forward and Backward Euler transient solution method. These should be replaced with an interface to Rythmos to gain access to a larger number of numerical solvers.
