std::vector< IRAD::Util::Parameters > &  componentParameters 
)

Initialization of the entire device data structure.

This function builds the device including:

  • reading input files
  • initializing components
  • loading physics modules
  • setting appropriate input data

We note that most of these things could be done in a constructor, but with the file reads, it is easier to do them outside the constructor to make error tracking easier.

We will name the components in the order that they are read (erdc[1-9]) correlating to their position in the componentParameter vector.

Parameters
deviceParametersParameter object defining the device parameters
componentParametersVector of Parameter objects defining the component parameters
Returns
int An error code
